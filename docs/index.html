Frequently Asked Questions

Do I need an API key?

Yes! Get a free API key from Pollinations.ai. Configure it in extension options (right-click ‚Üí Options). You can also choose your preferred language there. If you don't provide an API key in Options, the selected area will simply be copied to the clipboard.

Is this legal?

It depends on your school/university. ExamAssist is a tool - its use is your responsibility. We just provide the code. üòâ

Can teachers detect this?

The code is fully obfuscated and runs in an isolated space. Very hard to detect, but never 100% safe. Use with caution.

Which test platforms are supported?

Works on any website! However, TestPortal (testportal.pl, testportal.net) has special protection bypass to prevent detection of leaving the page. On other platforms, the extension works but without this detection bypass.

How can I add my own features?

It's open source! Fork on GitHub, add features and send a pull request. Let's make it better together!
